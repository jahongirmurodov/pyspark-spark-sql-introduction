## Sample Dataset

This repository includes a small sample dataset (`sample_data.csv`) created from a larger dataset (~2GB).

The sample contains the first rows of the original data and is provided for:
- testing the code
- demonstrating data structure
- quick local execution without heavy resources

The full dataset is not included due to its size.
